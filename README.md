# 🎓 LearnCey Online Learning Platform - MERN Stack MVC Web Application

A modern and feature-rich **Online Learning Platform** built using the **MERN Stack (MongoDB, Express.js, React, Node.js)** with **MVC architecture**. This application enables **Students**, **Tutors**, and **Admins** to interact within a secure and scalable online education ecosystem.

---

## 🚀 Features Overview

### 👨‍🎓 For Students
- Browse and **enroll** in available courses and class sessions
- **Rate and provide feedback** on courses
- **Receive email notifications** on enrollment and updates
- **Secure payments** for course/session purchases
- **Reset password** functionality via email
- Access course materials/documents (Cloudinary integration)

### 👩‍🏫 For Tutors
- Register and submit for **verification by Admin**
- After approval, **create and publish** courses and sessions
- **Earn income** from student enrollments (85% share)
- **Upload course content** securely via Cloudinary
- Conduct live sessions using **WebRTC-based video conferencing**

### 🛠️ For Admin
- **Verify tutors**, manage users
- Access **platform analytics** and reporting tools
- Monitor overall course and revenue activity

---

## 💡 Innovative Features
- ✉️ **Email Notifications** for enrollment, password reset, etc.
- 🔐 **Role-based access control (RBAC)** for Student, Tutor, and Admin
- 🔁 **Password reset via email token**
- 💳 **Secure Payment Gateway Integration**
- 📁 **File Uploading via Cloudinary** (for course documents and media)
- 🎥 **Live Class Sessions using WebRTC**
- 🧾 **Revenue Sharing** – Platform takes **15% commission**, Tutor earns **85%**

---

## 🧰 Tech Stack

### Frontend
- **React.js** with modern hooks and component-based architecture
- **Responsive UI** – mobile-friendly and modern design

### Backend
- **Node.js** with **Express.js**
- **MongoDB** for flexible and scalable NoSQL database
- **JWT Authentication** for secure login and session management
- **MVC Design Pattern** for clean separation of concerns

---

## 🔐 Security
- Role-based authorization (Admin, Tutor, Student)
- JWT token-based authentication
- Secure API endpoints and server-side validations
- Payment data handled securely via third-party integrations

---

## 📈 Revenue Model
- Each course or session purchase:
  - **15%** commission retained by the platform
  - **85%** paid to the **Tutor** automatically
